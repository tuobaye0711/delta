## 🏗️ 技术架构

### 前后端分离架构
```
┌─────────────┐
│   Browser   │
└──────┬──────┘
       │ HTTP
       ▼
┌─────────────┐
│  Frontend   │  Vue 3 + Vite + Element Plus
│  (Port 5173)│
└──────┬──────┘
       │ API
       ▼
┌─────────────┐
│   Backend   │  Express + Prisma
│  (Port 3000) │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  Database   │  SQLite
│  dev.db     │
└─────────────┘
```

### 数据模型
```
BulletType (弹种)
    ├── PriceHistory[] (价格历史)
    ├── UserTransaction[] (交易记录)
    └── UserTarget[] (目标价格)

UserTransaction (用户交易)
    └── BulletType (关联弹种)

UserTarget (目标价)
    └── BulletType (关联弹种)
```

## 🎓 使用教程

### 1. 查看实时价格
打开首页仪表盘，查看所有弹种的当前价格和涨跌幅。

### 2. 添加持仓
进入"持仓管理"页面，点击"添加交易记录"，输入买入信息。

### 3. 查看价格趋势
在弹种详情页，选择时间维度，查看价格历史曲线。

### 4. 设置目标价
在弹种详情页，点击"添加目标"，设置止盈/止损价格。

### 5. 查看持仓统计
在"持仓管理"页面，查看持仓总价值、未实现利润等统计数据。

## 💡 倒子弹策略

根据调研，推荐以下策略：

### 最佳买入时机
- 凌晨 2-4 点（价格低谷）
- 赛季初期（新弹种上市）
- 活动前（提前布局）

### 最佳卖出时机
- 晚上 8-10 点（玩家在线高峰）
- 双休日（需求增加）
- 赛季中期（价格高位）

### 风险控制
- 设置止损价格（建议 -10% 到 -15%）
- 分散投资（不要把所有资金投入单一弹种）
- 保持现金流（留足资金应对突发机会）
